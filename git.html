GIT

Git in here

Git is an interesting program. The point of git is to be able to use git commands, in terminal, git bash, or through vs code, in order to synch branches and projects locally from your computer to github website, and host the project in your repository. Personally, I got it to work for a good week or two, then there was problems. The idea behind git is that you are able to take projects anywhere and branch or test with multiple people. But for me it worked a while initially, and after I could not get vs code, my choice of editor, to recognize git. It would sometimes recognize “git status” without a $, but most of the time it was 100% failure even after multiple reinstalls. The idea of it and initial use though was good. By cloning copies from online to your desktop, branching, and loading it back online, you can test new ideas at drop of a hat and then forget them if they fail. Github also is just great for hosting projects as it lets you share folders and see ideas of others. Good if you need an idea or to reference unusual code.
