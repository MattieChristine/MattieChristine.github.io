UX

Have a pleasant Experience

The UX is the user experience, a back-end, behind the curtain scene for users that they never get to see but is vital in controlling how optimal and how well the webapp functions behind the scenes. This is the exact opposite of the UI, user interface, but they both play a vital role in web development. You need both the clean and easy to use UI so people can navigate the site, and the UX needs to be clean and crisp so that they can get function out of the website that they need. UX is more back-end type web development, server side coding, writing APIs, creating libraries, or work with other non-user facing portions of design. By controlling and editing the back-end, the user will have a better experience even if it is less seen and more felt. Elements of good UI and UX combined are key to making users want to come back to websites or use apps, because by being physically and functioningly appealing a user will likely come back and spread word, causing more traction to gain and your site or product to grow.
